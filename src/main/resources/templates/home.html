Bem-vindo ao Spring Boot!

O que é o Spring Boot

O Spring Boot é um framework que facilita a criação de aplicativos Java, eliminando a complexidade de configuração.

Com o Spring Boot, você pode configurar facilmente servidores web embutidos, banco de dados, segurança e muito mais, usando configurações automáticas. Isso torna o desenvolvimento mais ágil e simples.

Como Funciona o Controller no Spring Boot

No Spring Boot, os **Controllers** são responsáveis por receber as requisições HTTP, processá-las e retornar uma resposta. Eles são a parte central da arquitetura MVC (Model-View-Controller), onde o controller conecta o modelo e a visualização.

  • @Controller: Usado para definir um controlador que responde às requisições web e retorna visualizações.
  • @RestController: Combina @Controller e @ResponseBody, indicando que os métodos dentro do controller retornam dados diretamente para a resposta HTTP (usado em APIs REST).
  • @GetMapping, @PostMapping: Anotações usadas para mapear métodos HTTP a métodos de controlador específicos.

Características principais do Spring Boot

  • Autoconfiguração (Auto-Configuration): O Spring Boot configura automaticamente os componentes com base nas dependências presentes no projeto.
  • Produtividade e Convenção sobre Configuração: Oferece convenções que minimizam a necessidade de configurações complexas.
  • Spring Boot Starter POMs: O Spring Boot oferece starters para tarefas comuns como acesso a banco de dados, segurança e APIs REST.
